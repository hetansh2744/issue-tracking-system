Text-Based Issue Tracking System

By Hetansh – CPSC 3720 Project

About the Project

This project is a text-based Issue Tracking System (ITS) built in C++, using an MVC-style architecture with repositories, controllers, and a SQLite database. It supports creating, updating, deleting, tagging, and searching issues.

  • Create and manage issues
  • Add comments and assign users
  • Tag issues and filter by status
  • Backed by a SQLite database

Demo UI (Frontend Only)

This simple demo lets you enter issue details and “pretend” to create an issue. Right now it runs in the browser only (no backend connected). Later we can plug this into the real API.

Created Issues (In-Browser Demo)

These issues are stored only in your browser’s memory for demo purposes. Refreshing the page will clear them.
